## Headings
```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Paragraphs
```markdown
This is a paragraph.
```

## Line Breaks
```markdown
First line.  
Second line.
```

## Emphasis
```markdown
*Italic* or _Italic_

**Bold** or __Bold__

***Bold and Italic*** or ___Bold and Italic___
```

## Blockquotes
```markdown
> This is a blockquote.
```

## Lists

### Unordered List
```markdown
- Item 1
- Item 2
    - Subitem 1
    - Subitem 2
```

### Ordered List
```markdown
1. Item 1
2. Item 2
     1. Subitem 1
     2. Subitem 2
```

## Code

### Inline Code
```markdown
`inline code`
```

### Code Block
```markdown
```
code block
```
```

## Horizontal Rules
```markdown
---
or
***
or
___
```

## Links
```markdown
[Link Text](URL)
```

## Images
```markdown
![Alt Text](URL)
```

## Tables
```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
```
